# Technical Recruitment News

Dagelijks ophalen van recruitment trends en insights voor de technische arbeidsmarkt.

## Over dit project

Deze website toont actuele trends, vacature-analyses en arbeidsmarkt insights specifiek voor technisch recruitment in Nederland.

## Features

- Dagelijkse trend updates
- Vacature-analyse van 500+ bedrijven
- ICP bedrijven monitoring
- Concurrent intelligence
- Market insights dashboard

## Tech Stack

- Static HTML/CSS/JavaScript
- Deployed via Netlify
- Data pipeline via GitHub Actions

## Deployment

Deze site wordt automatisch gedeployed via Netlify wanneer er wijzigingen worden gepusht naar de main branch.

## Bronnen

Data wordt verzameld via:
- Intelligence Hub market scrapers
- Notion databases
- LinkedIn Newsletter Automation
